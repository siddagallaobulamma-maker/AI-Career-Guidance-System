# AI-Based Career Guidance System

## Description
This project uses Machine Learning to recommend suitable career options
based on student skills, interests, and academic performance.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Decision Tree Algorithm

## How to Run
1. Install required libraries:
   pip install pandas scikit-learn

2. Run the program:
   python career_guidance.py

## Sample Output
Recommended Career: Software Engineer
